Finance Dashboard

A simple and interactive finance dashboard built to track income, expenses, and financial insights.

---
Features:

- Dashboard Overview (Balance, Income, Expenses)
- Transactions Table with Search & Filter
- Role-Based UI (Admin / Viewer)
- Insights Section (Spending analysis)
- Responsive Design

---

 Tech Stack:
- React (Vite)
- JavaScript
- CSS
